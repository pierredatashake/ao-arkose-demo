# AO Arkose - Demo SEO/GEO Marseille Prado

Demo de re-traitement SEO/GEO de la page Arkose "Que faire a Marseille".

## Fichiers

- `index.html` : version SEO/GEO enrichie (Top 5 activites Marseille, sommaire, FAQ, auteur, comparatif, etc.)
- `index.original.html` : version d'origine pour comparaison
- `css/` : copie locale des CSS d'origine (le HTML reference le CDN Arkose en production)

## Elements SEO/GEO ajoutes

- Sommaire cliquable (11 ancres internes)
- Encadre "Answer First" (300 mots) pour citation LLM
- Section FAQ avec 6 questions (rich snippets ready)
- Bloc auteur avec photo, bio, expertise
- Tableau comparatif des 5 activites
- Dates publication + mise a jour
- Temps de lecture
- Boutons de partage social (haut + bas)
- Articles similaires recommandes
- Metadonnees visibles (categorie + tags)
- Zone commentaires et avis avec formulaire
